margin 요소의 내부 여백

1. margin 방향별 저장법

  • margin-top:n ; 위에만 n 적용
  • margin-right: n; 우측에만 n 적용
  • margin-bottom:n ; 아래에만 n 적용
  • margin-left:n ; 좌측에만 n 적용

2. margin 값 갯수별 저장법

  • margin: 5em; 양 사방 같은 값으로 동일하게 적용
  • margin: 5em 10%; 상하, 좌우 서로 같은 값으로
  • margin: 5em 10% 2em; 상,좌우, 하 순서
  • margin: 1em 10% 2em 30%;상,우,하,좌 순서로 각각 적용
