display

요소가 화면에 나타나는 방식(성격) 을 지정
  • display:inline; block인 요소를 inline 으로 지정
  • display:block; inline인 요소를 block으로 지정
  • display:inline-block ; 어떤 요소나 둘을 합친 방식을 지정
  • display:none ; 화면에서 사라짐 영역과 함께
inline 요소 - 크기, 상하 설정 적용이 불가함 한 줄에 나란히 옴
block 요소 - 크기, 상하 margin 적용 가능 한줄에 나란히 못 옴
inline-block 요소 - 크기 상하 margin 설정 적용이 가능하나 한 줄에 나란히 옴
