background

  • background-color: ; 배경 색상
    1. color명
    2. rgb(n,n,n)
    3. hsl(n,n,n)
    4. #nnnnnn
  • background-image:url("경로") ; 배경 이미지
    배경 이미지 두 개 지정은 url("경로1") , url("경로2")로 추가

background-image의 하위속성들: ;

  1. background-repeat: ; 배경이미지 반복 지정
    • repeat (기본값) 양사방 반복
    • repeat-x 가로로만 반복
    • repeat-y 세로로만 반복
    • no-repeat 반복하지 않음
  2. background-size: ; 배경이미지 크기 지정
    • cover 이미지 하나로 영역 전체를 다 덮기
      (이미지가 잘려나갈수도 있음)
    • contain 영역에 이미지 하나를 최대한 다 보이기
      (이미지가 잘려나가지 않음)
    • Xn Yn
      가로 세로 위치값을 지정
  3. background-position: ; 배경이미지 위치 지정
    • left/center/right top/center/bottom
      가로 세로 방향값을 조합
    • Xn Yn
      가로 세로 위치값을 지정
  4. background-attachment: ; 배경이미지 스크롤 여부 지정
    • scroll
    • fixed
    • local
