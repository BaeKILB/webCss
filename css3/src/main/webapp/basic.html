css basic

요소 () 가 화면에 나타나는 방식을 스타일 하는 언어

css 문법

선택자 {속성:속성값;}

css 대표 선택자 3가지

  • tag 선택자 - tag명을 그대로 선택자로 활용
  • id 선택자 - tag 에 id 를 추가하고 id 값을 선택자(#id)로 활용
  • class 선택자 - tag에 class를 추가하고 class값을 선택자(.class)로 활용

(* = 전체선택자)

css 적용방식 3가지

  • External CSS - 외부 스타일 소스가 별개의 파일로 존재 웹문서와 연결해서 적용 스타일 소스가 외부 파일론 존재 head 에 link 태그로 href 속성을 사용하여 불러옴
  • Internal CSS - 내부 . 내부파일 방식 스타일 소스가 웹 문서의 head 내에 존재 스타일 소스가 웹문서의 head 내에 (style type="text/css" ... /style)존재 스타일 내용이 페이지에 종속적임
  • Inline CSS - 일라인 방식 스타일 소스가 태그 바로 뒤에 존재. 코드가 길어지고 통일성 저해 적용 우선순위가 세 방식중 가장 높음
